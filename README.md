# 🚀 Smart IT Helpdesk System (ServiceNow)

## 📌 Overview

This project demonstrates a real-world IT Helpdesk system built using ServiceNow. It simulates how organizations manage IT incidents efficiently using automation and ITSM practices.

---

## 🧩 Features

* 🔹 Incident Management System
* 🔹 Auto Assignment using Flow Designer
* 🔹 Priority Calculation (Impact + Urgency)
* 🔹 SLA Configuration for time-based resolution
* 🔹 Email Notifications for incident lifecycle
* 🔹 Reports and Dashboard for performance monitoring

---

## ⚙️ Technologies Used

* ServiceNow Platform (CSA - Zurich)
* Flow Designer
* ITSM Concepts
